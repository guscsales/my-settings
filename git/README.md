# Git Aliases

### Install

#### Ubuntu
```bash
$ bash git/install/aliases.sh
```

### Aliases

#### git checkout "branch" 
```bash
$ git co feature/branch-name
```

#### git checkout master
```bash
$ git master
```

#### git branch
```bash
$ git br
```

#### git commit -m "message"
```bash
$ git ci -m "message"
```

#### git status
```bash
git st
```

#### git log -1 HEAD
```bash
git last
```

#### git log -1 reset HEAD -- file.js
```bash
git unstage file.js
```

#### git add .
```bash
git aa
```

#### git push origin -- branch
```bash
git po branch
```

#### git reset --soft HEAD^
```bash
git revertci
```
