# My settings

The settings how I use on my developing ecosystem. Include settings of vscode and git.

## Git Aliases

### Install
```bash
$ sh git-aliases.sh
```

### Aliases

#### git checkout "branch" 
```bash
$ git co feature/branch-name
```

#### git checkout master
```bash
$ git master
```

#### git branch
```bash
$ git br
```

#### git commit -m "message"
```bash
$ git ci -m "message"
```

#### git status
```bash
git st
```

#### git log -1 HEAD
```bash
git last
```

#### git log -1 reset HEAD -- file.js
```bash
git unstage file.js
```
